# Borrow Service

Borrow Service is a Spring Boot microservice for managing book borrowing and returning in a Library Management System.

## Requirements
- Java 17+
- Maven 3.9+

## Run
- Build: `mvn clean package`
- Start: `mvn spring-boot:run`

## Swagger
Swagger UI is available at `http://localhost:8083/swagger-ui.html` when the service is running.

## H2 Console
H2 Console: `http://localhost:8083/h2-console`
JDBC URL: `jdbc:h2:mem:testdb`
